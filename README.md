## Кейс 1: Инструкция

### Как создать событие в Яндекс.Календаре

**Зачем это нужно**  
Запланировать встречу или дедлайн с синхронизацией на все устройства.

**Перед началом**
- Аккаунт Яндекс (почта @yandex.ru)
- Открытый [Яндекс.Календарь](https://calendar.yandex.ru)

**Пошагово**
1. Откройте Яндекс.Календарь.
2. Нажмите на нужный день и время → появится окно «Новое событие».
3. Введите название (например, `Собеседование`).
4. Установите время начала и окончания.
5. Включите опцию `Напоминание` → выберите «За 15 минут».
6. Нажмите `Сохранить`.

**Совет**
Чтобы пригласить коллегу, нажмите «Пригласить» → введите его почту.
