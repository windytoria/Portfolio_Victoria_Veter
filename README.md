### Как создать событие в Яндекс.Календаре

**Зачем это нужно**  
Запланировать встречу с автоматическим напоминанием за 15 минут до начала.

**Перед началом**
- Открыть браузер с авторизацией в Яндекс (в правом верхнем углу — ваша почта)
- Открыть Яндекс.Календарь: перейдите на [calendar.yandex.ru](https://calendar.yandex.ru)

**Пошагово**

1. В календаре найдите нужную дату (например, 5 февраля).
2. Кликните левой кнопкой мыши на ячейку времени (например, «15:00») в основном поле календаря.
   → Откроется окно «Новое событие» справа.
3. В поле «Название» введите текст (обязательно!): например, `Собеседование`.
4. Убедитесь, что время установлено правильно:
   - «Начало»: 15:00
   - «Окончание»: 16:00
   *(Если нужно событие «весь день» — поставьте галочку «Весь день» под полями времени)*
5. Заполните поля (необязательно) «Название события» (например, «Совещание по проекту X») и «Место» (офис, конференц‑зал, ссылка на видеовстречу) 
6. Напоминание включено по умолчанию («За 15 минут»). Если нет — раскройте блок «Напоминание» и выберите из списка.
7. Чтобы пригласить коллег:
   * нажмите «Пригласить участников»;
   * введите email‑адреса (например, `ivanov@company.ru`);
   * выберите уровень доступа:  
     * «Может редактировать» (для организаторов);  
     * «Только просмотр» (для участников).
8. Нажмите синюю кнопку «Создать» внизу окна (справа).

**Что будет после**
- Событие появится в календаре цветной полосой.
- За 15 минут до начала придёт уведомление в браузере и на почту.

**Советы и рекомендации**
* Для повторяющихся событий: в окне «Новое событие» выберите «Повторять» → задайте периодичность (ежедневно, еженедельно и т. д.).
* Для видеовстреч: добавьте ссылку на конференцию в поле «Место» (например, Zoom, Яндекс Телемост).
* Чтобы изменить событие: кликните по нему в календаре → внесите правки → нажмите «Сохранить».
* Чтобы удалить событие: откройте его → нажмите «Удалить» → подтвердите действие.

**Если не получается**
* Окно «Новое событие» не открывается → обновите страницу (Ctrl + F5), проверьте, что вы вошли в нужный аккаунт.
* Не получается пригласить участников → убедитесь, что email‑адреса введены корректно, проверьте настройки доступа к календарю (раздел «Настройки» → «Доступ»).
* Событие не сохраняется → проверьте подключение к интернету, попробуйте другой браузер (Chrome, Firefox, Яндекс Браузер).
